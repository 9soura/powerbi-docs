	<title>Connect to Data in Power BI Designer</title>
	
	<div class="MsoNormal"><font face="Calibri" size="3">The Power BI Designer lets you easily connect to the ever
expanding world of data. If you don’t have the Power BI Designer, you can
download and install it from <b style="mso-bidi-font-weight: normal;">here</b>.
When you start the Power BI Designer, you’re presented with a new <b style="mso-bidi-font-weight: normal;">Report</b> view. </font></div><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">From the bottom left corner of the Power BI Designer,
select <b style="mso-bidi-font-weight: normal;">Query</b>.</font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span></span><span><img src="https://powerbi.uservoice.com/assets/79160274/QueryOverview_BlankPane.png"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">There are <i style="mso-bidi-font-style: normal;">all sorts</i>
of data sources available in the Query window. The following image shows how to
connect to data, by selecting the <b style="mso-bidi-font-weight: normal;">File</b>
ribbon, then <b style="mso-bidi-font-weight: normal;">Get Data &gt; More</b>.</font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span></span><span><img src="https://powerbi.uservoice.com/assets/79160739/PBI_Designer_GSG_GetData.gif"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">For this example, we'll connect to a <b style="mso-bidi-font-weight: normal;">Web</b> data source. </font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">Imagine you’re retiring – you want to live where there’s
lots of sunshine, preferable taxes, and good health care. Or… perhaps you’re a
data analyst, and you want that information to help your customers – as in,
help your raincoat manufacturing client target sales where it rains a <i style="mso-bidi-font-style: normal;">lot</i>. </font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">Either way, you find a Web resource that has interesting
data about those topics, and more:</font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><a href="http://www.bankrate.com/finance/retirement/best-places-retire-how-state-ranks.aspx"><i style="mso-bidi-font-style: normal;"><font color="#0563c1" face="Calibri" size="3">http://www.bankrate.com/finance/retirement/best-places-retire-how-state-ranks.aspx</font></i></a></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">You select <b style="mso-bidi-font-weight: normal;">Get Data
&gt; Web</b> and type the address.</font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span><img src="https://powerbi.uservoice.com/assets/76050540/ConnectToData_3.png"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">When you select <b style="mso-bidi-font-weight: normal;">OK</b>,
the <b style="mso-bidi-font-weight: normal;">Query</b> functionality of the Power
BI Designer goes to work. Query contacts the Web resource, and the <b style="mso-bidi-font-weight: normal;">Navigator</b> window returns the results of
what it found on that Web page. In this case, it found a table (Table 0) and
the overall Document. We’re interested in the table, so we select it from the
list. The <b style="mso-bidi-font-weight: normal;">Navigator</b> window displays
a preview.</font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span></span><span><img src="https://powerbi.uservoice.com/assets/79160451/DataSources_FromNavigatorDialog.png"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">At this point we can edit the query before loading the
table, by selecting <b style="mso-bidi-font-weight: normal;">Edit Query</b> from
the bottom of the window, or we can load the table. </font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">If we select <b style="mso-bidi-font-weight: normal;">Edit</b>, the table is loaded and the Query view is presented. The Query
Settings pane is displayed (if it’s not, you can select <b style="mso-bidi-font-weight: normal;">View</b> from the ribbon, then <b style="mso-bidi-font-weight: normal;">Show
&gt; Query Settings</b> to display the Query Settings pane). Here’s what that
looks like.</font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span></span><span><img src="https://powerbi.uservoice.com/assets/79160469/Designer_GSG_EditQuery.png"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">All those scores are text rather than numbers, and we need
them to be numbers. No problem – just right-click the column header, and select
<b style="mso-bidi-font-weight: normal;">Change Type &gt; Whole Number</b> to
change them. To choose more than one column, first select a column then hold
down <b style="mso-bidi-font-weight: normal;">SHIFT</b>, select additional
adjacent columns, and then right-click a column header to change all selected columns. Use <b>CTRL</b> to choose columns that are not adjacent. </font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span></span><span><img src="https://powerbi.uservoice.com/assets/79160481/Designer_GSG_ChangeDataType.png"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">In <b style="mso-bidi-font-weight: normal;">Query
Settings</b>, the Applied Steps will reflect any changes that were made. As you make additional changes to the data, Query will record those changes in the Applied Steps section, which you can adjust, revisit, rearrange, or delete as necessary. </font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span></span><span><img src="https://powerbi.uservoice.com/assets/79160541/Designer_GSG_AppliedSteps_ChangedType.png"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">Additional changes to the table can still be made after it’s
loaded, but for now this will do. When we select Report, the Power BI Designer
switches to <b style="mso-bidi-font-weight: normal;">Report</b> view, and the
data (as modified by the query steps) is loaded into the model. </font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span></span><span><img src="https://powerbi.uservoice.com/assets/79160658/Designer_ConnectToData_LoadQuery.png"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri" size="3">Once the table is loaded, the Query view in the Power BI
Designer lets you to shape the query, combine it with other data, or otherwise
transform it however you see fit. </font></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><span></span><span><img src="https://powerbi.uservoice.com/assets/79160661/Designer_GSG_EditQuery.png"></span><br></p><font face="Times New Roman" size="3">

</font><p style="margin: 0in 0in 8pt;"><font face="Calibri"><font size="3">Of course, this is a simple model with a single data
connection; most Power BI Designer reports will have connections to different
data sources, shaped to meet your needs, with relationships that produce a rich
data model.<span style="mso-spacerun: yes;">&nbsp; </span></font></font></p><font face="Times New Roman" size="3">
