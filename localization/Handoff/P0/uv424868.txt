	<title>Dashboards in Power BI</title>
	
	<div>

</div><p style="margin: 0in; font-family: Calibri; font-size: 11pt;">Dashboards, reports, and datasets are at the heart of Power BI. Create personalized&nbsp;<b><i>dashboards </i></b>to monitor your most important data, at a glance.&nbsp; A dashboard combines on-premises and cloud-born data in a single pane of glass, providing a consolidated view across the organization regardless of where the data lives. Each metric, or insight, is displayed on the dashboard as a <b><i>tile</i></b>.&nbsp; </p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><br></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;">Need help understanding the building blocks that make up Power BI?&nbsp; See <a href="http://support.powerbi.com/knowledgebase/articles/487029-power-bi-preview-basic-concepts">Power BI - Basic Concepts</a>.</p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><br></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;">When you first go to Power BI, you're looking at a sample<b> </b>dashboard for retail data. For detailed information about the sample, see <a href="http://support.powerbi.com/knowledgebase/articles/471112-sample-datasets">Sample datasets</a>.<br><br><br></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><span><img src="https://powerbi.uservoice.com/assets/82600174/dashboard.png"></span><br></p><span><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><br>Each tile is a snapshot of information taken from the underlying dataset.&nbsp; Learn how to <a href="http://support.powerbi.com/knowledgebase/articles/425669-tiles-in-power-bi">add tiles to a dashboard</a>. The tiles on a dashboard can be <a href="http://support.powerbi.com/knowledgebase/articles/424878-edit-a-tile-resize-move-rename-delete">resized, moved, deleted, and renamed</a>.&nbsp; </p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><br></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;">Unless you remove it, the sample dashboard is always available for you to use to explore and learn.&nbsp; <br><b><br></b></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><span style="font-family: sans-serif; font-size: 14px; font-style: normal; font-variant: normal; font-weight: bolder; background-color: rgb(249, 250, 251);">NOTE</span><span style="line-height: 19.6px; font-family: sans-serif; font-size: 14px; font-style: normal; font-variant: normal; background-color: rgb(249, 250, 251);">: The sample that comes with Power BI is read-only.&nbsp; For a writable version of this sample dashboard and report, in the navigation pane, click&nbsp;</span><span style="font-family: sans-serif; font-size: 14px; font-style: normal; font-variant: normal; font-weight: bolder; background-color: rgb(249, 250, 251);">Get Data &gt; Retail Analysis Sample</span><span style="line-height: 19.6px; font-family: sans-serif; font-size: 14px; font-style: normal; font-variant: normal; background-color: rgb(249, 250, 251);">.&nbsp;</span></p><h3>Create a dashboard</h3><p style="margin: 0in; font-family: Calibri; font-size: 11pt;">Learn how to <a href="http://support.powerbi.com/knowledgebase/articles/475163-create-a-power-bi-dashboard">create a new dashboard </a>and add tiles.</p>

<h3>Add tiles to a dashboard</h3><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><span><span><a href="http://support.powerbi.com/knowledgebase/articles/434354-get-data">Get data </a>and <a href="http://support.powerbi.com/knowledgebase/articles/425669-tiles-in-power-bi">add tiles </a>to your dashboard.<br>

</span></span></p><h3>Dashboard size</h3><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><span><span>Dashboards can be wider and longer than the browser window. If so, they have scroll bars. Because&nbsp;you want&nbsp;to show everything important at a glance, we recommend making the dashboard fit the browser window. For more tips, see <a href="http://support.powerbi.com/knowledgebase/articles/433616-tips-for-designing-a-great-dashboard">Tips for designing a great dashboard</a>.</span></span></p><h3>Working with Dashboards in the navigation pane</h3><p style="margin: 0in; font-family: Calibri; font-size: 11pt;">Power BI can have more than one dashboard -- as you add more dashboards, their titles are added to the list under the Dashboards heading.&nbsp; The dashboard you're currently viewing&nbsp;is highlighted. Only one dashboard can be open at a time.<br><span><br></span></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><span><img src="https://powerbi.uservoice.com/assets/82735066/dashboard%20pane%20piece.png"></span><br></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;">
