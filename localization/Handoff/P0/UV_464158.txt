Get the Power BI Designer

	<div class="MsoNormal">Before you can use some of the more advanced query, modeling, and report features in Power BI, you'll need to get the Microsoft Power BI Designer.</div><h2>Minimum requirements</h2><ul><li>Windows 7 / Windows Server 2008 R2, or later</li><li>.NET 4.5</li><li>Internet Explorer 10 or later</li></ul><b>Note: <h2>To download and install the designer</h2><ol><li>In Power BI, click the Downloads icon&nbsp;<span><img src="https://powerbi.uservoice.com/assets/82774660/DownloadButton.png"></span><span></span>&nbsp;<span></span>&nbsp;&gt; <b>Power BI Designer </b>(located in the top-right corner of the site page)</li><li>Click <b>Run</b>.<br></li></ol><span></span><blockquote><span><span><img src="https://powerbi.uservoice.com/assets/76123339/download.png"></span></span></blockquote><h2><span></span>With the designer, you can...</h2><h3>Get data</h3><p>The designer makes discovering data easy. You can import data from a wide variety of data sources. After you connect to a data source, you can shape the data to match your analysis and reporting needs. </p><h3>Create relationships</h3><p>When you&nbsp;import two or more tables, oftentimes you'll need to create relationships between those tables.&nbsp;The designer includes the Manage Relationships dialog, where you can use&nbsp;Autodetect to let the designer find and create any relationships, or you can create them yourself. Learn more at <a href="http://support.powerbi.com/knowledgebase/articles/464155-create-and-manage-relationships">Create and manage relationships</a>.</p><h3>Create reports</h3><p>The designer includes the Report View. Select the fields you want, add filters, choose from dozens of visuals.&nbsp;Report View&nbsp;gives you the same great report and visual tools just like when creating a report on PowerBI.com. Learn more at <a href="http://support.powerbi.com/knowledgebase/articles/461283-report-view-in-the-power-bi-designer">Report View in the Power BI Designer</a>.</p><h3>Save</h3><p>With the designer, you can save your work as a Power BI file.&nbsp;Power BI&nbsp;files have a .pbix extension.</p><blockquote><span><img src="https://powerbi.uservoice.com/assets/75846678/PBIDesignerSavedFile.png"></span></blockquote><h3><font face="Times New Roman" size="3">
</font>Upload reports</h3><p>You can upload reports your&nbsp;created and&nbsp;saved in the designer to your Power BI&nbsp;site. To learn more, see <a href="http://support.powerbi.com/knowledgebase/articles/461278-upload-power-bi-designer-reports">Upload Power BI Files</a></p><p><br></p><p><span><span style='background: rgb(249, 250, 251); color: black; font-family: "Arial",sans-serif; font-size: 10.5pt;'></span></span><br></p><div></div>
