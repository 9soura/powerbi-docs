	<title>Excel data in Power BI</title>
	
	<p>You can use any XLSX Excel workbook&nbsp;- Excel 2007, 2010, and 2013 - as a dataset for your Power BI dashboards and reports. <a href="https://support.powerbi.com/knowledgebase/articles/597003">Get your Excel data into Power BI</a>.</p><p><span style="line-height: 1.4em;">&nbsp;For the best results, your Excel workbook data:</span><br></p>
<ul>
<li>is in table format. (

<a href="https://support.office.com/en-us/article/Create-an-Excel-table-in-a-worksheet-e81aa349-b006-4f8a-9806-5af9df0ac664?ui=en-US&amp;rs=en-US&amp;ad=US" target="_blank">Instructions for formatting Excel data as a table</a>)</li>

<li>is prepared as an Excel&nbsp;data model (typically created using Power Pivot). Don't worry if you don't know how to do this, it's not required.</li>
<li>has columns marked by data type (e.g, date, number). For more information:</li>

<ul><li><a href="https://support.office.com/en-US/Article/Data-Types-Supported-in-Data-Models-e2388f62-6122-4e2b-bcad-053e3da9ba90" target="_blank">data types in Excel data models</a></li>
<li><a href="https://support.office.com/en-us/article/Format-numbers-f27f865b-2dc5-4970-b289-5286be8b994a" target="_blank">formatting number data types in Excel</a></li></ul></ul><h2>What is imported?</h2><p>Power BI brings in&nbsp;the data in the workbook from worksheet tables&nbsp;or from the Excel data model. Data model content takes precedence&nbsp;over worksheet content.</p><p>Power BI brings in Power View sheets, but doesn't yet recognize PivotTables or PivotCharts or custom fields.</p><h3>Data types</h3><p>Power BI recognizes values and supports these data types: Whole Number, Decimal Number, Currency, Date, True/False, Text).&nbsp; Marking data as specific data types in Excel will improve the Power BI experience.</p><h2>Refreshing Excel data</h2><ul><li>When you upload an Excel workbook from your computer, you can't refresh the data.&nbsp;</li><li>When you connect to an <a href="http://support.powerbi.com/knowledgebase/articles/471009-excel-data-stored-in-onedrive">Excel workbook in OneDrive</a>, you can <a href="http://support.powerbi.com/knowledgebase/articles/474669-refresh-data-in-power-bi">refresh the data</a>.</li></ul><h2>Troubleshooting</h2><p>Currently,&nbsp;when connecting to Excel,&nbsp;Power BI only imports data that is part of a named table or a data model. As a result:</p><ul><li>When importing a workbook, if there are no named tables, Power View sheets, or Excel data models in the workbook, you see this error. The article "<a href="https://support.powerbi.com/knowledgebase/articles/501937--we-couldn-t-find-any-data-in-your-excel-workbook">We couldn't find any data in your Excel workbook</a>" explains how to fix the workbook and reimport it.</li></ul>
