	<title>Microsoft Dynamics CRM content pack for Power BI</title>	<span style="line-height: 1.4em; font-family: Calibri; font-size: 11pt;">The Microsoft
Dynamics CRM content pack for Power BI allows you to easily access and analyze
your data. The&nbsp;</span><span style="font-family: Calibri; font-size: 14.6666669845581px; font-style: normal; font-variant: normal; line-height: 20.533332824707px;">content pack&nbsp;</span><span style="line-height: 1.4em; font-family: Calibri; font-size: 11pt;">uses the OData feed to create a descriptive model,
with all the entities and&nbsp;measures needed such as Accounts, Activities,
Opportunities, Product, Leads, Users and more.</span><div style="direction: ltr;"><span></span>
</div><p style="margin: 0in; font-family: Calibri; font-size: 11pt; direction: ltr;">&nbsp;</p><div style="direction: ltr;">

</div><p style="margin: 0in; font-family: Calibri; font-size: 11pt; direction: ltr;">The data is filtered
down to opportunities that were created or closed in the last 365 days and will
be refreshed daily.</p><div style="direction: ltr;">

</div><p style="margin: 0in; font-family: Calibri; font-size: 11pt; direction: ltr;">&nbsp;</p><p lang="en-CA" style="margin: 0in; font-family: Calibri; font-size: 11pt;">Requirements:</p><ul type="disc" style="margin-top: 0in; margin-bottom: 0in; margin-left: 0.375in; unicode-bidi: embed; direction: ltr;"><li style="vertical-align: middle;"><span style="font-family: Calibri; font-size: 11pt;">You need to specify     valid OData URL for a Dynamics CRM Online instance (the&nbsp;<span style="font-family: Calibri; font-size: 14.6666669845581px; font-style: normal; font-variant: normal; line-height: 20.533332824707px;">content pack&nbsp;</span>will not work with&nbsp;an on-premises CRM version).</span></li><li style="vertical-align: middle;"><span lang="en-CA" style="font-family: Calibri; font-size: 11pt;">An     administrator must enable the OData endpoint in the site settings. The     address of the OData endpoint can be found by navigating to <b>Microsoft Dynamics CRM &gt; Settings &gt; Customizations &gt; Developer Resources</b>.&nbsp; The OData URL is listed under <b>Service Endpoints</b>.&nbsp;&nbsp;It typically has     the format: </span><a href="https://[instance_name].crm.dynamics.com/XRMServices/2011/OrganizationData.svc"><span lang="en-US" style="color: black; font-family: Calibri; font-size: 11pt;">https://[instance_name].crm.dynamics.com/XRMServices/2011/OrganizationData.svc</span></a><span lang="en-US" style="font-family: Calibri; font-size: 11pt;"> </span></li><li style="vertical-align: middle;"><span lang="en-US" style="font-family: Calibri; font-size: 11pt;"></span><span lang="en-US" style="font-family: Calibri; font-size: 11pt;">The user account/identity that you use to access Dynamics CRM Online must be the same as the one you are signed up for using with Power BI. When logging into CRM Online, you will be auto-signed into CRM with the same identity you are using for Power BI. If you wish to sign into CRM Online with a different account, please register as a Power BI user using the same account. We hope to resolve this issue in an upcoming release. </span></li></ul>
