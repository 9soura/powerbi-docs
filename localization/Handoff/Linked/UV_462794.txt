GitHub content pack for Power BI

	<p style="margin: 0in; font-family: Calibri; font-size: 11pt;">The GitHub content pack for Power BI allows you to gain insights into a GitHub repository (also
known as repo) with data around contributions, issues, pull requests and active
users.</p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><br></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;">Connect to the <a href="https://app.powerbi.com/getdata/services/github">GitHub content pack </a>for Power BI.</p>

<p style="margin: 0in; font-family: Calibri; font-size: 11pt;">&nbsp;</p>

<p style="margin: 0in; font-family: Calibri; font-size: 11pt;">Creating this connection retrieves your data and automatically&nbsp;provides a dashboard and related reports based on the data. </p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><span style="font-weight: bold;"><br></span></p><p style="margin: 0in; font-family: Calibri; font-size: 11pt;"><span style="font-weight: bold;">Requirements</span>:</p>

<li><div style="margin: 0in; font-size: 11pt;"><span style="font-family: Calibri;">Permission granted to the Power BI for GitHub app during first login</span></div></li>
